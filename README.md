# install-node

通过脚本在CentOS和Ubuntu上安装NodeJS和Yarn，并配置阿里云镜像加速。

## 给脚本可执行权限
```bash
# CentOS
sudo chmod 777 node-centos.sh

# Ubuntu
sudo chmod 777 node-ubuntu.sh
```

## 执行脚本
```bash
# CentOS
./node-centos.sh

# Ubuntu
./node-ubuntu.sh
```

> 注意：CentOS系统执行完脚本之后，需运行`source /etc/profile`刷新环境变量。
